# Automated Testing Project for Saucedemo.com

## Описание проекта
Этот проект содержит автоматизированные тесты для демо-сайта [saucedemo.com](https://www.saucedemo.com/). 
Тесты покрывают основные функциональности: авторизацию, работу с корзиной и API.

## Технологический стек
- **Python 3.9** - язык программирования
- **Pytest** - фреймворк для тестирования
- **Selenium** - автоматизация браузера
- **Requests** - работа с HTTP запросами
- **GitHub Actions** - CI/CD пайплайн
